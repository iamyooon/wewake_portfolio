# D) 월 1,100만 원 인출의 **복리 저해 효과(Compounding Penalty)** — 정밀 계산
## D-1) 2026~2030 인출이 2030 자산에 만든 손실(최종 CAGR 14% 기준)

- 미래가치 합계 = 2026 인출×(1.14^4) + 2027×(1.14^3) + 2028×(1.14^2) + 2029×(1.14^1) + 2030×(1.14^0)
