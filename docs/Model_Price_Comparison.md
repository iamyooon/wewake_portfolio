## 4. 보고서 1회 생성 시 대략 비용 (참고)

- **가정:** Step1(Grok) 50K in / 20K out, Step2(Gemini) 60K in / 25K out, Step3(OpenAI) 80K in / 40K out (공통 프롬프트·초안·감사 포함).
- **Grok 4.1 Fast:** 0.05×0.20 + 0.02×0.50 ≈ **$0.02**  
- **Gemini 3 Flash:** 0.06×0.50 + 0.025×3.00 ≈ **$0.11**  
- **OpenAI gpt-5.2:** 0.08×1.75 + 0.04×14 ≈ **$0.72**  
- **합계(대략):** **약 $0.85/회** (gpt-5.2 사용 시).  
- 동일 토큰에 **gpt-5.2-pro** 사용 시 Step3만 약 **$8~10** 수준으로 증가 가능.

### ⚠️ gpt-5.2 Reasoning(Thinking) 토큰 주의

- gpt-5.2는 **reasoning_effort: medium**으로 내부 "Thinking" 토큰을 생성함.
- **Reasoning 토큰은 output으로 청구**되나, 응답 텍스트에 포함되지 않음.
- 스크립트가 과거에는 **가시적 출력만**으로 추정해 **실제 비용의 1/10 수준**으로 과소 추정했음.
- **실제 검증:** [OpenAI Usage](https://platform.openai.com/usage)에서 일별·모델별 사용량 확인 권장.

## 4-2. 기존 구조 vs 2라운드 협의 구조 비용 차이 (예측)

**기존 구조 (현재):** 3회 API 호출  
- 1라운드: Grok(초안+Alpha) → Gemini(감사+Beta) → GPT(최종)

**제안 구조 (2라운드 협의 + 3라운드 GPT):** 5회 API 호출  
- 1라운드: Grok(초안+Alpha) → Gemini(감사+Beta)  
- 2라운드: Grok(수용+반박만) → Gemini(수용+반박만)  
- 3라운드: GPT(합의·대립 정리 + 자신 의견 → 최종 리포트)

| 구분 | 기존 (3회) | 제안 (5회) | 비고 |
|------|------------|------------|------|
| **호출 수** | 3 | 5 | +2 (Grok R2, Gemini R2) |
| **Grok** | 50K in / 20K out | 50K + **25K in / ~4K out** (R2) | R2는 "수용+반박만"이라 출력 짧음 |
| **Gemini** | 60K in / 25K out | 60K + **~8K in / ~4K out** (R2) | R2 입력=Grok R2 응답 |
| **OpenAI** | 80K in / 40K out | **~100K in** / 40K out | R1+R2 내용 모두 전달 가정 |

**토큰 가정 (제안 구조):**  
- Grok R2: 25K in (Gemini 감사 전문), 4K out  
- Gemini R2: 8K in (Grok R2 + 지시), 4K out  
- GPT R3: 100K in (R1 초안·감사 + R2 수용·반박), 40K out  

**비용 추정 (1M tokens당: Grok $0.20 in / $0.50 out, Gemini $0.50 / $3, OpenAI $1.75 / $14):**

| 항목 | 기존 | 제안 | 차이 |
|------|------|------|------|
| Grok | 0.05×0.20 + 0.02×0.50 = **$0.02** | + 0.025×0.20 + 0.004×0.50 ≈ **$0.007** | +**$0.007** |
| Gemini | 0.06×0.50 + 0.025×3 = **$0.11** | + 0.008×0.50 + 0.004×3 ≈ **$0.017** | +**$0.017** |
| OpenAI | 0.08×1.75 + 0.04×14 = **$0.72** | 0.10×1.75 + 0.04×14 ≈ **$0.74** | +**$0.02** |
| **합계/회** | **약 $0.85** | **약 $0.89** | **+약 $0.04~0.05** |

**요약:**  
- 제안 구조는 **1회당 약 $0.04~0.05 (약 5~6%)** 비용 증가 예상.  
- 호출 2회 증가·R2 출력은 짧게 유지하면 **Grok·Gemini 비용이 상대적으로 작게** 늘고, GPT 입력만 다소 늘어나는 구조라 전체 증가폭은 제한적.
