# OpenAI gpt-5.2-pro 404 오류 안내

## 결론: **설정은 잘못된 게 아닙니다**

`.env`의 API 키, 스크립트 인자 등 **당신이 한 설정은 문제 없습니다.**  
404는 **API 키/계정이 유효하다는 전제** 아래, “이 요청에 쓰인 **모델**을 이 계정으로 쓸 수 없다”는 뜻입니다.

---

## 404가 나는 흔한 이유 (OpenAI 문서/커뮤니티 기준)

1. **모델 접근 권한**
   - `gpt-5.2-pro`가 **Models API**(`/v1/models`) 목록에는 보이지만,  
     **Chat Completions**(`/v1/chat/completions`)에서는 **계정/플랜별로 사용 불가**일 수 있음.
   - 특정 티어·비용 한도·베타 접근이 필요한 모델일 수 있음.

2. **모델 ID 차이**
   - 문서상 Chat Completions용 **스냅샷**은 예: `gpt-5.2-pro`, `gpt-5.2-pro-2025-12-11`.
   - `gpt-5.2-pro`가 404라면, 같은 제품의 **다른 스냅샷**(예: `gpt-5.2-pro-2025-12-11`)은 동작할 수 있음.

3. **계정/요금**
   - 무료 티어는 **gpt-5.2-pro 미지원**(문서에 “Free – Not supported” 명시).
   - 결제 수단·한도·리전에 따라 특정 모델만 허용될 수 있음.

즉, **설정을 잘못해서가 아니라**,  
“이 계정으로는 `gpt-5.2-pro`(또는 그 이름의 Chat Completions 엔드포인트)를 쓸 수 없다”는 의미에 가깝습니다.

---

## 당신이 할 수 있는 점검

1. **OpenAI 플랫폼**
   - https://platform.openai.com/settings/organization  
   - 사용량·요금제·제한 확인.
2. **결제/한도**
   - https://platform.openai.com/settings/billing  
   - 결제 수단, 크레딧, 한도 확인.
3. **모델 접근**
   - https://platform.openai.com/playground  
   - 플레이그라운드에서 **gpt-5.2-pro** 선택 후 호출해 보기.  
   - 여기서도 404/권한 오류면, 계정 권한/플랜 문제일 가능성이 큼.

---

## 스크립트 쪽 대응 (이미 반영됨)

- **기본 요청 모델**: 그대로 `gpt-5.2-pro` 사용.
- **폴백 순서**:  
  `gpt-5.2-pro` → **(추가)** `gpt-5.2-pro-2025-12-11` → `gpt-4o` → …  
  그래서 `gpt-5.2-pro`가 404여도, 같은 제품의 스냅샷이나 gpt-4o로 자동으로 이어서 동작합니다.

정리하면, **설정 실수가 아니라 “이 계정으로 gpt-5.2-pro 사용이 안 되는 상황”**이고,  
위 점검 + 스크립트 폴백으로 처리되도록 되어 있습니다.
