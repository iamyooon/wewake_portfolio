# gpt-5.2-pro 비용 안내

## 왜 한 번 질의에 돈이 많이 나갔나?

**gpt-5.2-pro**는 **토큰당 단가가 매우 높은** reasoning 모델입니다.

| 구분 | gpt-5.2-pro (공식 기준) |
|------|--------------------------|
| Input | **$21.00 / 1M 토큰** |
| Output | **$168.00 / 1M 토큰** |

- **16,741 토큰** 사용 시 대략:
  - 전부 output 가정: 16,741 × (168/1,000,000) ≈ **$2.81**
  - input 5k + output 11,741 가정: 약 **$2.1**
- 질의 한 번에 1만 토큰대만 써도 **$2~3** 수준이라, 크레딧이 적으면 “한 번에 다 썼다”처럼 느껴질 수 있습니다.

---

## 본 스크립트에서 OpenAI가 쓰는 구간

보고서 한 번 생성 시 **OpenAI는 2번** 호출됩니다.

1. **초안 작성**: 긴 프롬프트(portfolio_prompt 전체) + 긴 응답(보고서 초안) → 수천~1만 토큰대
2. **최종 작성**: 초안+리뷰+검증 전체 + 긴 응답(최종 보고서) → 1만 토큰대 이상 가능

그래서 **gpt-5.2-pro**를 쓰면 보고서 한 번에 **수만 토큰**이 나가고, **$5~10 이상** 나갈 수 있습니다.

---

## 비용 줄이는 방법

1. **기본 모델을 gpt-4o로 변경**  
   - gpt-4o는 훨씬 저렴 (input/output 단가 낮음).  
   - 스크립트 기본값을 `gpt-4o`로 두고, 필요할 때만 `--openai-model gpt-5.2-pro` 로 지정.

2. **gpt-5.2-pro는 “꼭 필요할 때만”**  
   - 일상적인 보고서 생성은 gpt-4o, 중요한 검토나 최종본만 gpt-5.2-pro 사용.

3. **사용량·한도 확인**  
   - https://platform.openai.com/usage  
   - https://platform.openai.com/settings/billing  
   에서 사용량·잔액·한도 확인 후, 필요하면 예산 알림 설정.

---

원하면 스크립트 **기본 OpenAI 모델을 gpt-4o로 바꿔서** 비용을 줄이도록 수정해 줄 수 있습니다.
