# AI별 시스템 프롬프트 (Step 1~3)

스크립트 실행 시 **시스템 프롬프트**는 이 폴더의 MD 파일에서 읽습니다.  
파일이 없거나 읽기 실패 시 스크립트 내 폴백 문구를 사용합니다.

| 파일 | 용도 |
|------|------|
| `step1_grok_system.md` | Grok — 데이터 분석관 (Alpha CAGR 예측) |
| `step2_gemini_system.md` | Gemini — 리스크 감사관 (Beta CAGR 예측) |
| `step3_openai_system.md` | OpenAI — 수석 매니저 (최종 CAGR 확정) |

- **내용:** MD 본문만 사용됩니다. 첫 줄이 `# 제목`이어도 그대로 API에 전달됩니다.
- **인코딩:** UTF-8.
- **수정:** 이 파일을 편집하면 다음 실행부터 적용됩니다. 스크립트 수정 없이 프롬프트만 바꿀 수 있습니다.
